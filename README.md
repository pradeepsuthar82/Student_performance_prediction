# Student Performance Prediction

## Overview

This project uses Machine Learning techniques to predict a student's final grade based on demographic, social, and academic factors. The model analyzes historical student data and identifies key factors that influence academic performance.

## Problem Statement

Educational institutions often need to identify students who may require additional support. By predicting final grades, educators can take proactive measures to improve student outcomes and academic success.

## Dataset

The dataset contains information about 395 students and 33 features related to:

- Student demographics
- Family background
- Study habits
- School support
- Internet access
- Previous academic performance

### Target Variable

- **G3** – Final Grade

### Important Features

- G1 (First Period Grade)
- G2 (Second Period Grade)
- Study Time
- Absences
- Failures
- Parent Education
- Internet Access
